🔒 Secure PyQt6 desktop app for Claude.ai

Finally, a way to keep Claude in a box (literally). For those whose company 
blocks AI development tools - this gives you the freedom to get the benefits 
from the AI age while staying secure.

✅ Blocks malware downloads
✅ Validates URLs
✅ Denies creepy permissions
Security-hardened because "trust but verify" is for IT permission givers.

Made for engineers in restricted environments. Open source (MIT).
